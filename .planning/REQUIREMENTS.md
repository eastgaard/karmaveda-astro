# Kravspecifikation

## Funktionelle Krav
- **Frontend**: Hurtig rendering via Astro.
- **Styling**: Brug Tailwind CSS til alt design.
- **Deployment**: Applikationen skal kunne bygges og deployes til Cloudflare Pages.

## Ikke-funktionelle Krav
- **Performance**: Lighthouse score skal minimeres til nær 100 på desktop og mobil.
- **Udvikling**: Det lokale udviklingsmiljø skal køre i IDX uden ekstra manuel opsætning (port 4321).
