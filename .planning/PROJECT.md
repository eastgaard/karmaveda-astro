# Astro App med Cloudflare Pages

Dette projekt er en moderne, lynhurtig frontend bygget med Astro, Tailwind CSS og optimeret til at køre på Cloudflare Pages (SSR/Hybrid).

## Målsætning
At etablere en solid base for et webprojekt med optimal ydeevne og en gnidningsfri deployment-pipeline via GitHub til Cloudflare Pages.

## Teknologistak
- **Framework**: Astro
- **Styling**: Tailwind CSS
- **Deployment**: Cloudflare Pages (@astrojs/cloudflare)
- **Udviklingsmiljø**: Google IDX (.idx/dev.nix konfigureret)
