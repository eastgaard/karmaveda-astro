# Phase 3 Context: Udarbejd grundlæggende layout og forside

<decisions>
- **Design System & Tema**: Lyst tema centreret omkring lysegrønne og brune jordfarver for et organisk og naturligt udtryk.
- **Layout Struktur**: En fast top-navigation (header) med indbygget understøttelse af dropdown-menuer.
- **Forsidens Indhold**: Opbygning der matcher den visuelle og tematiske identitet (Ayurveda/yoga/wellness), inspireret direkte af brandet.
</decisions>

<specifics>
- **Inspiration**: Tydelig reference til `www.karmaveda.dk` (Pga. blokeret adgang til siden via bot, vil designet baseres generelt på lyse/grønne/brune naturfarver og et "premium wellness" look, indtil yderligere brand assets tilføjes).
</specifics>
